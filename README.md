## Instalación
# Clona el repositorio
```bash
git clone https://github.com/felipeperezleal/tripster_ma.git
```

# Ingresa al directorio del proyecto
```bash
cd tripster_ma
```

# Instala las dependencias
```bash
flutter pub get
```

# Ejecuta la aplicación
```bsah
flutter run
```
